# Digital Clock Project

A simple web-based digital clock created for my web class assignment. 

## Description
This project displays a real-time digital clock in a browser. The main goal was to practice using HTML, CSS for centering elements, and JavaScript to handle the time logic.

## Features
* Displays current time (Hours:Minutes:Seconds)
* Perfectly centered layout (Vertical and Horizontal)
* Clean, minimalist design

## How to Run
1. Download the project files.
2. Open `index.html` in any web browser.

## Technologies Used
* HTML
* CSS (Flexbox for centering)
* JavaScript (`setInterval` function for the clock)

## ## 🔒 Security Note
This is a front-end educational project. 
- Client-Side Only: All logic runs in the user's browser; no user data is collected or sent to a server.
- Dependencies: This project uses no external libraries or APIs, making it safe from third-party vulnerabilities.
